## Getting Started localhost
First, clone repo
``` 
git clone https://github.com/Pgsalbuquerque/DreamFit
```

Second, install the dependencies:
```bash
npm install
# or
yarn
````

Third, run the development server:

```bash
npm run dev
# or
yarn dev
```
